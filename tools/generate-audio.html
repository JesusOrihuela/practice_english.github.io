🔊 Audio Generator

Pre-generates Kokoro TTS audio files for all topics. Run this locally whenever you add new phrases.

1 — Output Folder

Select the shared/audio/ folder in your project. Files will be written directly to disk — no downloads needed.

2 — Voices

Each phrase gets one audio file per voice selected. Grades from the official VOICES.md — higher = more natural.

3 — Speed

0.95 is the sweet spot — natural prosody without rushing. Dictation playback rate is handled separately in the app.

0.95

4 — Sources

Phrase topics generate {topic}/{i}_{voice}.wav. Vocabulary generates vocab/{i}_{voice}.wav from words.json.

5 — Generate

The model loads once (~92 MB, cached by browser). Existing files are skipped automatically — safe to re-run after adding new phrases.
